# Screenshots Directory

This folder contains application screenshots for documentation.

## File Naming Convention:
- `01-login.png` - Login page (with GitHub link animation!)
- `02-dashboard.png` - Main dashboard
- `03-inventory.png` - Inventory management page
- `04-inventory-add.png` - Add inventory item modal
- `05-inventory-detail.png` - Item detail with image
- `06-loans.png` - Loan management page
- `07-loans-create.png` - Create new loan modal
- `08-consumables.png` - Consumables tracking page
- `09-analytics.png` - Analytics and reports page (top section)
- `10-analytics-2.png` - Analytics page (scrolled down view)
- `11-cannibalization.png` - Cannibalization tracking page
- `12-cannibalization-add.png` - Add cannibalization modal
- `13-docker-containers.png` - Docker Desktop with running containers

## Guidelines:
- Format: PNG (high quality)
- Resolution: 1920x1080 or full screen
- Show both light and dark theme (optional)
- Hide sensitive data if any
